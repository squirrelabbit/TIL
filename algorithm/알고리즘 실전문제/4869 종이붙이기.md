가로합 = 30(input) / 세로합 = 20인 경우의 수

## 코드

```python
def paper(n):
    if n == 1:
        return 1
    elif n == 2:
        return 3
    return paper(n-1) + (2 * paper(n-2))

T = int(input())
for tc in range (1, T+1):
    n = int(int(input())/10)
    
    print("#{} {}".format(tc, paper(n)))
```
